## 项目简介
本项目依托于复旦大学“曦源项目”，针对门诊患者的心理量化特征（焦虑抑郁指数）展开了一系列研究，本仓库展示了我们团队的研究过程以及得到的成果，具体来说：
1. 我们在相关医院的耳鼻喉门诊跟诊了大量的患者，制作出了ZZHL-25焦虑抑郁医患对话数据集demo，数据集包含了133份门诊场景下患者与医生的对话，同时对话涵盖了初诊与复诊场景，尽可能实现了场景的全覆盖，同时对于每位收集到数据的患者，在门诊结束后同意填写了我们准备的衡量焦虑抑郁指标PHQ-9和GAD-7的问卷，最后经过数据清洗，我们将把这部分工作开源，这是目前国内少有的门诊医疗场景下的心理焦虑抑郁指数相关数据集。
2. 我们对比研究了不同AI方法在情绪风险识别上与门诊医疗场景的契合度，具体来说，我们对比了zero-shot learning，few-shot learning+特征工程，以及传统特征工程+机器学习的方案。同时我们基于现有的医疗大模型，我们简单对比了Lora微调方案与使用VADER，Personality，LIWC-CH等方法提取患者语言学特征，将特征提取出来之后将量化指标融入提示词，再给医疗大模型进行预测的方案。
3. 作为医学研究，我们研究了心理指数在门诊阶段实时趋势，利用模型监测家长在哪个阶段焦虑感突然上升。我们将包含了初诊，复诊阶段的门诊记录人为划分，同时将同一段门诊记录划分为三个阶段：主诉期、告知期、咨询期。最终制作时间序列折线图，横轴是对话进程，纵轴是焦虑得分，尝试找到医生话语对于患者焦虑心情的影响。同时我们对比研究了不同心理特征区间患者的语言学特征以及人口学特征（如城乡差异）的相关关系。
4. 在项目上，我们最终开发了一套针对门诊环境的焦虑抑郁智能识别系统。
**特别的**，我针对医疗场景复杂的特点开发了一套数据处理管线，直接实现从自然语言提取和自然语言处理的一系列流程，实时挖掘门诊患者的心理量化特征，利用机器学习/深度学习模型实现焦虑与抑郁风险的自动化、精准化识别。在未来，我们会考虑使用Agent工作流将其他的相关工作融入我们的工作。

## 技术架构
以微调后中文医疗大语言模型（BERT/GPT类）为基础，融合多模态特征工程（声学、本土化LIWC词典等），搭配小样本学习与专家规则修正。优化模型推理速度，通过隐私脱敏机制（语音转特征向量/抽象标签）保障数据安全合规。
## 量化指标
核心量表（PHQ-9 和 GAD-7），体现研究的科学依据。
